# 🚀 Talent Path - MERN Job Portal

A modern, full-stack job portal application built with the MERN stack (MongoDB, Express, React, Node.js) that streamlines job searching, recruitment, and application management.

## ✨ Features

- **Job Listings**: Browse and search through available job opportunities
- **Job Applications**: Apply to jobs with a seamless user experience
- **Recruiter Dashboard**: Manage job postings and view applications
- **User Profiles**: Create and customize professional profiles
- **Authentication**: Secure user authentication with JWT
- **Email Notifications**: Automated email updates for applications and status changes
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Real-time Updates**: Dynamic UI with instant feedback

## 🛠️ Tech Stack

- **Tailwind CSS 4.3.2** - Utility-first CSS framework

- **Framer Motion 12.42.2** - Animation library
